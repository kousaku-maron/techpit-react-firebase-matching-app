# 環境構築

* 下記ファイルを作成。(自分用テンプレ)
```
.eslintrc.json
.prettierrc.json
tsconfig.json
webpack.config.js
```

* パッケージの作成
```
yarn init
```

* 必要なライブラリインストール
```
yarn add ....
```

* `package.json`にコマンド追加
```
scripts
husky
lint-staged
```

* `.gitignore`の作成
* vscodeの設定ファイル追加
* typesの追加

* アプリのエントリーファイル作成
```
src/index.html
src/index.tsx
src/App.tsx
```

* アプリの起動
```
yarn dev
```
